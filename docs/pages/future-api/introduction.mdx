# Stackflow Future API

- 초기 로딩 성능 최적화를 위해 Stackflow의 인터페이스를 개편하고 있어요.
- React 의존성 없는 Loader API 지원을 추가합니다.

## Future API?

- Future API는 Stackflow 2.0 API의 미리보기 버전이에요.
- 완전히 확정된 API는 아니기 때문에 사용해보시면서 불편함을 제보해주시면 충분히 추가적인 기여가 가능해요.
- 내부 동작의 변화 없이 API만 바뀌었기 때문에 기존 API 역시 사용이 가능해요. 기존 API는 2.0 버전 업데이트 후 삭제될거에요.

## 최소 패키지 버전

신규로 추가된 Future API를 사용하려면 아래 패키지로 업데이트해주세요.

- `@stackflow/core`: 1.1.0 버전
- `@stackflow/react`: 1.4.0 버전
- `@stackflow/plugin-history-sync` : 1.7.0 버전
- (신규) `@stackflow/config`: 1.2.0 버전
